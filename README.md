# Hapi Socket-Proxy test

# Install
```shell
$ cd app && npm install
$ cd api && npm install
```

# Usage

You need [node-compose](https://www.npmjs.com/package/node-compose) to be installed.

```shell
$ node-compose
node-compose $ start all
```

Open your browser at [http://localhost:8090](http://localhost:8090).
